BlueX Description:
BlueX is an Open Source Java based Explorer cum Environment so that removable drives viruses can be detected. Since it works in the JVM memory so there is no threat from Autorun viruses as they will not be activated.

Secondly it has been backed up with several tools:
-Automatic Scanning
-Full furnished explorer with copying, deleteing and many more other features
-Advanced File Searching
-It provides plugin based environment so that multiple plugins can be installed
-Automatic updations (Proxy setting enabled)
-Plugin Management
-Exception Management
-Intellegent Threading Mechanism
-Favorites Management
-All new Help Center
-and many more..

Note:
Even you can create plugins for BlueX with the BlueX Plugin API. If you are really interested in developing Plugins for BlueX please contact me. Developing a plugin for BlueX is very simple. You just need to Implement one Plugin Interface.

Even you can integrate your made project with BlueX so that your project can be distributed around the world with the BlueX Plugin Central. No modification in your project is required for making it the BlueX plugin.
If you are interested please do contact me.
